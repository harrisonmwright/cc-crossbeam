## SECTION 1 — ADMINISTRATIVE CORRECTIONS

### Item 1 — Lot Area Inconsistency *(High Confidence)*

**Code:** Austin LDC §25-1-21 — lot area must be determined by licensed survey; all permit documents must reflect the same verified measurement.

The permit application (page 4) states a lot area of **7,330 sq ft**. The site development data table states **7,730 sq ft**. These values differ by 400 sq ft and appear within the same submittal package.

**Required Action:** Revise all submittal documents to reflect a single, consistent lot area value reconciled to the recorded plat or a current licensed survey. Note that both stated values exceed the 5,750 sq ft threshold — the governing HOME phase (Phase I, §25-2-773) applies regardless. However, the correct lot area is critical for accurate Subchapter F FAR calculations (see Items 3 and 4).

---

### Item 2 — Use Type Must Be Designated as Two-Unit Residential *(High Confidence)*

**Code:** HOME Phase I, Ord. 20231207-001, §25-2-773 — Duplex, Two-Unit, and Three-Unit Residential uses govern multi-unit development on SF-1, SF-2, and SF-3 lots. Austin LDC §25-2-901 (Accessory Apartments) was repealed by Ord. 20231207-001.

The submittal title blocks describe the project as "House w/ Habitable Attic" with a separate ADU, using framing consistent with the former Accessory Apartment use under §25-2-901. That use type was repealed by HOME Phase I. Two dwelling units on a single SF-3 lot — regardless of whether one is described as a primary residence and one as an ADU — now constitute **Two-Unit Residential use** under §25-2-773.

**Required Action:** Revise all title blocks, the permit application, and the Subchapter F GFA exhibits to explicitly designate the use type as **Two-Unit Residential** per §25-2-773. Confirm that all §25-2-773 development standards (setbacks, coverage, FAR limits) are being applied to both structures as a combined Two-Unit Residential development.
