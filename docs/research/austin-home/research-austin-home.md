# Austin, Texas — HOME Initiative Research

> **Research date**: March 2026
> **Subject**: Austin's HOME (Home Options for More Efficiency) Initiative — Phase 1 and Phase 2 Land Development Code amendments
> **Purpose**: Reference document for understanding Austin's approach to residential density reform; comparison context for CrossBeam's California ADU work

---

## Overview

Austin's HOME Initiative is a series of amendments to the City of Austin Land Development Code (LDC) that effectively ends exclusionary single-family zoning citywide. The initiative is split into two phases, each addressing a different dimension of the housing supply problem.

**The core shift:** HOME allows more units per lot (Phase 1) and more lots per parcel (Phase 2). Together, they enable missing-middle housing at scale across all SF-zoned land in Austin.

---

## Phase 1 — More Units Per Lot

### What It Does

- **Allows up to 3 housing units** (including tiny homes) by right on any SF-zoned property (SF-1, SF-2, SF-3)
- **Revises two-unit regulations** — updates development standards that apply when a property has two housing units
- **Removes the unrelated adults restriction** — eliminates the occupancy limit on the number of unrelated adults living in a housing unit

### Scope

Phase 1 applies specifically to:
- Duplex use
- Two-unit residential use
- Three-unit residential use

### Key Takeaway

**3 units by right on every SF-zoned lot in Austin.** No discretionary review, no conditional use permit — if the lot is SF-1, SF-2, or SF-3, you can build up to 3 units as a matter of right.

---

## Phase 2 — Smaller Lots, More Parcels

### What It Does

Phase 2 introduces a new land use category: **"small lot single-family residential use"** in SF-1, SF-2, and SF-3 districts.

**The core change:** Lowers the minimum residential lot size from **5,750 sq ft** (standard) to **1,800 sq ft**, enabling one unit to be built on a much smaller parcel.

### Eligible Lot Size

| Condition | Min Lot Size |
|-----------|-------------|
| Standard SF residential | 5,750 sq ft |
| Small lot single-family (HOME Phase 2) | 1,800 sq ft |

A small lot single-family use requires a lot of **at least 1,800 sq ft but less than 5,750 sq ft**.

### Application Types

Two paths to utilize Phase 2 regulations:

#### 1. Subdivision Applications (for lots ≥ 5,750 sq ft)
Most existing lots exceed the standard 5,750 sq ft minimum. To take advantage of Phase 2 on these lots, the owner must first subdivide.

**Residential Infill Subdivision** — a new streamlined subdivision process created specifically for HOME Phase 2. To qualify:
- Site must not exceed 1 acre
- Must be a resubdivision/replat (not a new subdivision of raw land)
- Must include only land that was already platted as a residential subdivision
- Must not require a plat vacation

#### 2. Building Permit Applications (for lots already < 5,750 sq ft)
Existing lots that are already below the standard minimum can skip subdivision and go straight to a building permit. These lots apply HOME Phase 2 development regulations directly to establish small lot single-family residential use.

### Key Takeaway

Phase 2 enables **infill development on smaller parcels** — you can subdivide a standard SF lot into multiple smaller lots and build a unit on each. Combined with Phase 1, an owner could subdivide a single 11,500 sq ft lot into two 5,750 sq ft lots and put 3 units on each, for **6 units total** where one house stood before.

---

## Combined Effect

| Before HOME | After HOME Phase 1 + 2 |
|-------------|----------------------|
| 1 unit max on SF lot | Up to 3 units by right |
| 5,750 sq ft minimum lot | 1,800 sq ft minimum lot |
| Occupancy limits on unrelated adults | No occupancy restriction |
| Discretionary review for density | Ministerial (by-right) approval |

## Comparison to California ADU Law

Austin's HOME Initiative and California's ADU statutes (Gov. Code §§ 66310–66342) are tackling the same problem — housing supply in single-family zones — but through different mechanisms.

| Dimension | Austin HOME | California ADU Law |
|-----------|-------------|-------------------|
| **Approach** | Rezoning / LDC amendment | State preemption of local zoning |
| **Units allowed** | Up to 3 by right (Phase 1) | ADU + JADU + primary = up to 3 units (§ 66323) |
| **Minimum lot size** | 1,800 sq ft (Phase 2) | No minimum lot size for ADUs (§ 66314) |
| **Approval process** | Ministerial (by-right) | Ministerial, 60-day approval (§ 66317) |
| **Owner-occupancy** | No requirement | Required for JADUs (if shared sanitation) |
| **Short-term rental** | No restriction in HOME itself | JADUs prohibited from STR < 30 days (AB 1154) |
| **Fee exemptions** | N/A (Texas has no equivalent) | No impact fees for ADUs ≤ 750 sq ft (§ 66311.5) |
| **Enforcement** | City-level (no state mandate) | HCD enforcement; non-compliant ordinances null and void |

### Key Differences

1. **Scale of reform**: HOME eliminates single-family exclusivity outright. California ADU law grafts density onto existing SF zoning without changing the underlying zone.

2. **Enforcement backstop**: California has state-level HCD enforcement — cities that don't comply lose their local ordinance and must apply state law. Austin HOME is purely local; if a future council reverses it, there's no state backstop.

3. **Lot subdivision**: HOME Phase 2 enables lot splitting as part of the reform. California's SB 9 (Gov. Code § 66411.7) is the rough equivalent — allows splitting SF lots into two parcels — but it's more constrained than Austin's approach.

4. **ADU vs. primary use**: California ADU law creates an *accessory* use layered on the primary dwelling. Austin HOME treats additional units as primary uses (two-unit residential, three-unit residential), which has different implications for financing, insurance, and permitting.

## Relevance to CrossBeam

Austin's HOME Initiative is useful context for CrossBeam for several reasons:

1. **Market signal**: Cities across the country are moving toward by-right multi-unit zoning. Austin is the largest U.S. city to do this. CrossBeam's California focus is in the same policy direction — the question is not *whether* this happens but *when* the permitting infrastructure catches up.

2. **Permit complexity**: HOME Phase 2's subdivision + building permit two-step creates exactly the kind of complexity CrossBeam is designed to solve — multiple processes, multiple departments, multiple approval pathways. Texas expansion would be a natural fit for this tool.

3. **Comparison selling point**: For California homeowners and designers who feel state ADU law is restrictive, Austin is a useful benchmark: California is not behind — in some ways (fee exemptions, state enforcement, JADU framework) it's actually more developed than Austin's approach.
