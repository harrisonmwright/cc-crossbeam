# Rainey Street Density Bonus

## At a Glance
| Field | Value |
|-------|-------|
| LDC Section | §25-2-736 (Rainey Street Subdistrict of the Downtown Austin Plan) |
| Ordinance | No. 20130815-105 (Downtown Austin Plan); Rainey Street subdistrict adopted contemporaneously |
| Incentive Type | Density Bonus (within Rainey Street subdistrict of Downtown Austin Plan) |
| Applicability | Properties within the Rainey Street subdistrict, generally bounded by Red River Street, Cesar Chavez Street, IH-35 frontage road, and Town Lake/Lady Bird Lake |

## Incentives Granted
| Incentive | Standard Value | Bonus Value |
|-----------|---------------|-------------|
| Maximum height | 60 ft base in Rainey subdistrict | Up to 96 ft with affordability compliance |
| FAR | 8:1 base | Up to 10:1 |
| Setbacks | Rainey subdistrict standards | Front: 0 ft at designated frontages; side/rear per subdistrict |
| Ground floor | Active use required | Active use required; Rainey Street character preservation overlay applies |
| Parking | Per Downtown Plan standards | Reductions available; no parking minimums in subdistrict |

## Affordability Requirements
| Parameter | Rental | Ownership |
|-----------|--------|-----------|
| Set-aside % | 10% of total residential units | 10% of total residential units |
| Max income limit | ≤80% MFI | ≤80% MFI |
| Affordability period | 40 years | 40 years |
| Fee-in-lieu | Available at DSD/HCD-set rate |

**Math check:** `affordable_units / total_residential_units ≥ 0.10`

## Dimensional Standards Under Bonus
| Parameter | Rainey Street Standard |
|-----------|----------------------|
| Maximum height | 96 ft above natural grade |
| Maximum FAR | 10:1 |
| Front setback | 0 ft (build-to line at designated frontages) |
| Side/rear setback | Per Rainey subdistrict standards |
| Ground floor active use | Required at Rainey Street frontage; min 75% of frontage length |
| Parking minimum | None (Downtown Plan area) |
| Parking maximum | Per Downtown Plan transit-priority area rules |

## Required Plan Elements
1. **Cover sheet:** "Rainey Street Density Bonus" stated; LDC §25-2-736 cited; Rainey Street subdistrict confirmed
2. **Project data table:** Total residential units, affordable units, set-aside %, income limit, affordability period, tenure type; FAR calculation
3. **Affordability schedule:** Each affordable unit by ID, type, MFI tier, period
4. **Ground floor plan:** Active use shown at Rainey Street frontage; Rainey Street character preservation features (consistent with subdistrict's historic bar/restaurant character) noted
5. **Site plan:** Build-to-line compliance; height labeled; FAR calculation exhibit
6. **Floor plans:** Affordable units individually labeled
7. **Elevations:** Maximum height from natural grade; ground floor active use shown on frontage elevation

## Key Compliance Checks
- [ ] Property within Rainey Street subdistrict of Downtown Austin Plan
- [ ] LDC §25-2-736 cited on cover sheet
- [ ] Affordable count ≥ 10% of residential units — recompute from plan data
- [ ] Income limit ≤ 80% MFI

## Notes
- The Rainey Street subdistrict has overlay design standards intended to preserve the pedestrian-scale, bar/restaurant character of the street; purely architectural design compliance is beyond this skill's objective-standard scope — flag for [REVIEWER: verify Rainey Street overlay design standards]
- No parking minimum applies in the Rainey subdistrict (Downtown Plan area); any parking provided must comply with design standards
- Projects in the Rainey subdistrict should also comply with Downtown Austin Plan Community Benefits system; see `downtown-density-bonus.md` for context
- View corridor restrictions from the Downtown Austin Plan may apply to taller buildings; flag for [REVIEWER: verify view corridor compliance]
- Lady Bird Lake impervious cover restrictions may apply to lots near the shoreline; flag for [REVIEWER: verify waterfront overlay compliance]
