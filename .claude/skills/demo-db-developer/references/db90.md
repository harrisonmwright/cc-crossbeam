# DB90 — Density Bonus 90

## At a Glance
| Field | Value |
|-------|-------|
| LDC Section | §25-2-652 |
| Ordinance | No. 20190613-154 (adopted), amended Ord. 20230601-054 |
| Incentive Type | Density Bonus |
| Applicability | Lots in CS, CS-1, GR, LO, GO, MF-3, MF-4, MF-5, MF-6 base districts (and -V combining districts) within city limits; lot must be in a density bonus combining district (-DB90) approved via site plan |

## Incentives Granted
| Incentive | Standard Value | Bonus Value |
|-----------|---------------|-------------|
| Maximum height | Base district limit (typically 40–60 ft) | Up to 90 ft (approximately 7–9 stories) |
| FAR | Base district limit | Waived (no maximum FAR cap under DB90) |
| Setbacks | Base district minimums | Not modified by DB90; base district setbacks apply unless also in VMU or other overlay |
| Parking | Base district requirement | Not modified by DB90 (parking reductions require separate VMU or TOD combining district) |

## Affordability Requirements
| Parameter | Rental | Ownership |
|-----------|--------|-----------|
| Set-aside % | 12% of total residential units | 10% of total residential units |
| Max income limit | ≤80% MFI | ≤80% MFI |
| Affordability period | 40 years | 40 years |
| Fee-in-lieu | Available; rate set by City Council annually (contact HCD for current rate) |
| Alternate: deeper affordability | 5% at ≤50% MFI instead of 12% at ≤80% MFI (rental) | Not available |

**Math check formula:** `affordable_units / total_units ≥ 0.12` (rental) or `≥ 0.10` (ownership)

Example: 125 units × 12% = 15.0 → minimum 15 affordable units required (round up if fractional).

## Dimensional Standards Under Bonus
| Parameter | DB90 Standard |
|-----------|--------------|
| Maximum height | 90 ft above natural grade |
| Maximum FAR | None (waived) |
| Minimum setbacks | Base district standards apply; DB90 does not modify |
| Lot coverage | Base district standards apply |
| Impervious cover | City-wide impervious cover limits apply (25-8 subchapter) |
| Minimum lot area | None specific to DB90; base district applies |

## Required Plan Elements
The following must appear on the submitted plan set to support a DB90 review:
1. **Cover sheet:** Program name "DB90" explicitly stated; LDC §25-2-652 cited; zoning designation including "-DB90" suffix shown
2. **Project data table:** Total residential units, affordable unit count, set-aside %, income limit (MFI tier), affordability period, tenure type (rental/ownership)
3. **Affordability exhibit/schedule:** Sheet or table listing each affordable unit by unit number/ID, floor, unit type, AMI tier, and affordability period
4. **Site plan:** Building footprint, height dimension to highest point, lot area, setbacks labeled
5. **Floor plans:** All floors with individual units labeled; affordable units distinctly identified (e.g., "Affordable — 80% MFI")
6. **Elevations:** Maximum height dimension from natural grade shown on at least one elevation

## Key Compliance Checks
- [ ] Zoning designation includes "-DB90" combining district
- [ ] LDC §25-2-652 cited on cover sheet
- [ ] Affordable unit count ≥ 12% of total units (rental) or ≥ 10% (ownership) — recompute from plan data, do not trust stated percentage
- [ ] Income limit ≤ 80% MFI stated for all affordable units
- [ ] Affordability period ≥ 40 years stated
- [ ] Proposed height ≤ 90 ft (measured from natural grade per §25-1-21)
- [ ] Affordable units labeled individually on floor plans
- [ ] Affordability exhibit unit count matches cover sheet count
- [ ] If fee-in-lieu claimed: fee documentation present (do not compute amount — reviewer blank)

## Notes
- DB90 requires the -DB90 combining district be approved through a zoning case before a site plan can invoke the bonus; the skill does not re-verify zoning eligibility
- Mixed-use projects with hotel or commercial: only residential units count toward the unit denominator and the affordable set-aside
- Affordable unit mix requirement: affordable units must reflect the overall unit mix of the project (e.g., if 20% of total units are 2BR, approximately 20% of affordable units should be 2BR) — LDC §25-2-652(D)
- "Total units" for set-aside calculation excludes hotel keys and short-term rental units; confirm with project data table
- If the project also has -VMU combining district, VMU affordability requirements may differ — DB90 governs if both apply and DB90 is more restrictive
