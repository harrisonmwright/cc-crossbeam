# Smart Housing — Greenfield Single-Family

## At a Glance
| Field | Value |
|-------|-------|
| LDC Section | §25-1-501 through §25-1-519; §25-2-492 (SF district provisions) |
| Ordinance | No. 991202-J (Smart Housing base); SF-specific provisions in Ord. 20031211-H |
| Incentive Type | Development Incentive (fee waivers, lot size reduction, expedited review) |
| Applicability | New single-family residential subdivisions (greenfield development) on SF-zoned land; minimum 10-lot subdivision; requires City Council Smart Housing designation |

## Incentives Granted
| Incentive | Value |
|-----------|-------|
| Minimum lot size | Reduced to 2,500 sq ft (from standard 5,750 sq ft SF-3 minimum) |
| Lot width | Reduced to 25 ft (from standard 50 ft SF-3 minimum) |
| Front setback | Reduced to 15 ft (from 25 ft standard) |
| Side setback | Reduced to 5 ft (from 5 ft standard — unchanged in most SF districts) |
| Development fee waiver | City fees waived per Smart Housing program |
| Expedited review | Priority processing |
| Max lot coverage | 50% (increased from 40% standard) |

## Affordability Requirements
| Parameter | Value |
|-----------|-------|
| Set-aside % | 10% of lots/units in the subdivision |
| Max income limit | ≤80% MFI for ownership |
| Affordability period | 5 years (minimum); 10 years for enhanced incentives |
| Tenure type | Ownership only (Smart Housing Greenfield SF is an ownership program) |
| Fee-in-lieu | Not available |

**Math check:** `affordable_units / total_units ≥ 0.10` (applied at subdivision level, not individual lot level)

## Dimensional Standards Under Bonus
| Parameter | Smart Housing Greenfield SF Standard |
|-----------|-------------------------------------|
| Minimum lot area | 2,500 sq ft |
| Minimum lot width | 25 ft |
| Minimum front setback | 15 ft |
| Minimum side setback | 5 ft |
| Minimum rear setback | 10 ft |
| Maximum lot coverage | 50% |
| Maximum height | 35 ft (base SF district limit; not modified) |
| Parking | 1 covered space per unit (not waived) |

## Required Plan Elements
1. **Cover sheet:** "Smart Housing Greenfield — Single-Family" stated; LDC §25-1-501 cited; City Council resolution number; subdivision name
2. **Project data table:** Total lots, affordable lots/units, set-aside %, income limit, affordability period
3. **Plat/subdivision plan:** Lot dimensions for each lot; reduced lot sizes labeled; compliance with 2,500 sq ft minimum shown
4. **Affordable lot designation:** Specific lots designated as affordable identified on plat (lot numbers, addresses if assigned)
5. **Site plan (typical lot):** Reduced setbacks labeled; lot coverage calculation; parking shown
6. **Floor plan (representative unit):** Square footage, bedroom count, room labels

## Key Compliance Checks
- [ ] City Council Smart Housing resolution number cited
- [ ] Subdivision meets 10-lot minimum
- [ ] Affordable lot count ≥ 10% of total lots — recompute from plat data
- [ ] Income limit ≤ 80% MFI for ownership
- [ ] Affordability period stated (minimum 5 years)
- [ ] All reduced-standard lots meet 2,500 sq ft minimum area
- [ ] All reduced-standard lots meet 25 ft minimum width
- [ ] Front setback ≥ 15 ft on all lots
- [ ] Lot coverage ≤ 50%
- [ ] Parking: 1 covered space per unit shown on typical lot plan
- [ ] No fee-in-lieu claimed (not available)
- [ ] Affordable lots labeled on plat

## Notes
- Smart Housing Greenfield SF is a subdivision-level program, not a single-permit program; the plan set reviewed is typically a subdivision plat + civil engineering plans for the full subdivision, not an individual building permit
- Individual building permits for homes on Smart Housing lots are reviewed separately under standard SF code
- The 5-year affordability period is much shorter than most other programs; the plan set must clearly state the period
- City Council Smart Housing designation is a prerequisite; if the resolution number is absent from plans, flag as FAIL immediately
- Lot size reductions apply to lots designated as affordable AND to market-rate lots within the Smart Housing subdivision; verify which lots carry reduced standards
