# Affordability Unlocked

## At a Glance
| Field | Value |
|-------|-------|
| LDC Section | §25-2-656 |
| Ordinance | No. 20190613-153 |
| Incentive Type | Density Bonus (by-right, no combining district required) |
| Applicability | Any lot in SF-1 through SF-6, MF-1 through MF-6, and most commercial base districts; does NOT apply in PUD districts or on lots with deed restrictions inconsistent with the program |

## Incentives Granted
| Incentive | Standard Value | Bonus Value |
|-----------|---------------|-------------|
| Maximum height | Base district limit | 6 stories (no absolute foot limit specified — story-count governed) |
| FAR | Base district limit | 3.0 FAR for residential; 4.0 FAR if 100% affordable |
| Setbacks | Base district minimums | Reduced: front 5 ft, side 0 ft (adjacent to another lot in program), rear 5 ft |
| Minimum lot size | Base district requirement | Waived |
| Parking | Base district requirement | Waived entirely for projects ≥50% affordable; reduced for others |
| Unit type restrictions | Base district rules | Waived |

## Affordability Requirements

Affordability Unlocked has tiered affordability options. The developer selects a tier; higher tiers unlock greater bonuses.

| Tier | Min % Affordable | Max Income | Affordability Period | Bonus Unlocked |
|------|-----------------|------------|---------------------|----------------|
| Tier 1 | 50% of units | ≤60% MFI | 40 years | Standard bonuses |
| Tier 2 | 75% of units | ≤60% MFI | 40 years | 4.0 FAR, full parking waiver |
| Tier 3 | 100% of units | ≤60% MFI (≤80% for 10% of units) | 40 years | Max height, full setback waiver |

**Math check:** For Tier 1: `affordable_units / total_units ≥ 0.50`. For Tier 2: ≥ 0.75. For Tier 3: = 1.00.

Fee-in-lieu is **not available** under Affordability Unlocked — all affordable units must be on-site.

## Dimensional Standards Under Bonus
| Parameter | Standard |
|-----------|---------|
| Maximum height | 6 stories above grade (no maximum feet stated in ordinance) |
| Maximum FAR | 3.0 (Tier 1/2); 4.0 (Tier 3 / 100% affordable) |
| Front setback | 5 ft minimum |
| Side setback | 0 ft permitted if adjacent lot also in program; otherwise 5 ft |
| Rear setback | 5 ft minimum |
| Parking | Waived for Tier 3; reduced per §25-6 parking table for Tier 1/2 |
| Lot coverage | No maximum specified; impervious cover limits still apply |

## Required Plan Elements
1. **Cover sheet:** Program name "Affordability Unlocked" stated; LDC §25-2-656 cited; tier selected (Tier 1, 2, or 3) identified
2. **Project data table:** Total units, affordable units by tier, % affordable, income limit, affordability period (40 years), tenure type
3. **Affordability schedule:** Each affordable unit listed with unit ID, type, tier, MFI limit, affordability period
4. **Site plan:** Setbacks labeled with dimensions matching the tier's reduced standards; parking count shown (or notation of waiver)
5. **Floor plans:** All units labeled; affordable units individually identified by unit number and MFI tier
6. **Elevations:** Story count labeled or floor-to-floor heights noted to confirm ≤6 stories

## Key Compliance Checks
- [ ] Tier declared on cover sheet (Tier 1, 2, or 3)
- [ ] Affordable unit count meets or exceeds tier threshold (50% / 75% / 100%) — recompute from counts
- [ ] Income limit ≤60% MFI for all affordable units (or ≤80% MFI for up to 10% of units in Tier 3 only)
- [ ] Affordability period ≥ 40 years
- [ ] Proposed height ≤ 6 stories
- [ ] FAR ≤ 3.0 (Tier 1/2) or ≤ 4.0 (Tier 3)
- [ ] Setbacks match tier: front ≥ 5 ft, rear ≥ 5 ft, side per tier
- [ ] No fee-in-lieu claimed (not available)
- [ ] Parking: verify against tier (waived for Tier 3; standard reduced table for Tier 1/2)
- [ ] Affordable units individually labeled on floor plans

## Notes
- Affordability Unlocked is a by-right program — no combining district zoning case required; any eligible property can invoke it directly through a site plan or permit
- The program is frequently combined with HOME Initiative on SF-zoned lots; if combined, both sets of requirements apply independently
- 40-year affordability period is per §25-2-656(E); enforced via recorded affordability covenant (covenants must be executed before site plan approval)
- Projects must file an Affordability Unlocked Checklist with DSD at time of site plan submittal
- Height is story-based, not foot-based — plan must show story count or floor-to-floor dimensions to verify compliance
