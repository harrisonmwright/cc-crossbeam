# DB-ETO — Density Bonus East 11th and 12th Street

## At a Glance
| Field | Value |
|-------|-------|
| LDC Section | §25-2-655 |
| Ordinance | No. 20161215-121 |
| Incentive Type | Density Bonus (geographic overlay — East 11th and 12th Street Neighborhood Conservation Combining District) |
| Applicability | Properties within the East 11th Street and East 12th Street NCCD; lots must be within the NCCD boundaries as shown on the adopted NCCD map |

## Incentives Granted
| Incentive | Standard Value | Bonus Value |
|-----------|---------------|-------------|
| Maximum height | NCCD base limits (typically 40–60 ft depending on subarea) | Up to 65 ft within NCCD with affordability commitment |
| FAR | NCCD base limits | Increased FAR up to 3.0 with bonus |
| Parking | Standard §25-6 requirements | Reduced; some reductions available depending on proximity to transit corridor |
| Setbacks | NCCD base standards | Not modified by DB-ETO; NCCD standards apply |

## Affordability Requirements
| Parameter | Rental | Ownership |
|-----------|--------|-----------|
| Set-aside % | 10% of total residential units | 10% of total residential units |
| Max income limit | ≤80% MFI | ≤80% MFI |
| Affordability period | 40 years | 40 years |
| Fee-in-lieu | Available at HCD-approved rate |

**Math check:** `affordable_units / total_units ≥ 0.10`

## Dimensional Standards Under Bonus
| Parameter | DB-ETO Standard |
|-----------|----------------|
| Maximum height | 65 ft above natural grade |
| Maximum FAR | 3.0 |
| Setbacks | NCCD subarea standards apply (not modified by DB-ETO) |
| Parking | Per transit proximity table in §25-2-655(D) |
| Lot coverage | NCCD standards apply |

## Required Plan Elements
1. **Cover sheet:** "DB-ETO" or "Density Bonus East 11th/12th" stated; LDC §25-2-655 cited; NCCD subarea identified
2. **Project data table:** Total residential units, affordable units, set-aside %, income limit, affordability period, tenure type
3. **NCCD compliance note:** Subarea designation, applicable NCCD standards
4. **Affordability schedule:** Each affordable unit by ID, type, MFI tier, period
5. **Site plan:** Height labeled; FAR calculation shown; setbacks labeled per NCCD subarea
6. **Floor plans:** Affordable units individually labeled
7. **Elevations:** Maximum height from natural grade shown

## Key Compliance Checks
- [ ] Property is within East 11th/12th Street NCCD (zoning designation includes NCCD)
- [ ] LDC §25-2-655 cited on cover sheet
- [ ] NCCD subarea identified
- [ ] Affordable count ≥ 10% of total residential units — recompute from plan data
- [ ] Income limit ≤ 80% MFI
- [ ] Affordability period ≥ 40 years
- [ ] Proposed height ≤ 65 ft from natural grade
- [ ] Proposed FAR ≤ 3.0
- [ ] Setbacks comply with NCCD subarea standards (not DB-ETO itself)
- [ ] Affordable units individually labeled on floor plans
- [ ] If fee-in-lieu: documentation present; do not compute amount

## Notes
- DB-ETO is geographically restricted to the East 11th and 12th Street NCCD; properties outside this NCCD cannot use this program
- The NCCD has multiple subareas with varying standards; the subarea designation determines base height and setback limits that DB-ETO adds onto
- Historic compatibility review may apply in portions of the NCCD; flag for [REVIEWER: verify compatibility review not required]
- DB-ETO height bonus applies above the NCCD base limit, not above the citywide base district limit; confirm which limit applies to this subarea
- Compatibility standards from §25-2 Subchapter B may apply for properties abutting residential zoning
