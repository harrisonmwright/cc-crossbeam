# Micro-Unit Density Bonus

## At a Glance
| Field | Value |
|-------|-------|
| LDC Section | §25-2-654 |
| Ordinance | No. 20141211-203 |
| Incentive Type | Density Bonus |
| Applicability | Lots in MF-3, MF-4, MF-5, MF-6, CS, CS-1, GR, GO, LO base districts; project must consist primarily of micro-units (units ≤400 sq ft net rentable area) |

## Incentives Granted
| Incentive | Standard Value | Bonus Value |
|-----------|---------------|-------------|
| Maximum FAR | Base district limit | 25% increase over base district FAR |
| Maximum height | Base district limit | No additional height bonus (FAR bonus only) |
| Parking | Per §25-6 residential rate | Reduced: 0.5 spaces per micro-unit (instead of standard 1.0–1.5) |
| Setbacks | Base district standards | Not modified |

## Affordability Requirements
| Parameter | Value |
|-----------|-------|
| Set-aside % | 10% of total residential units |
| Max income limit | ≤60% MFI (stricter than most programs) |
| Affordability period | 40 years |
| Unit type requirement | Affordable units must be micro-units (≤400 sq ft) |
| Fee-in-lieu | Not available |

**Math check:** `affordable_units / total_units ≥ 0.10`

Note: The 10% affordable units must themselves be micro-units; they cannot be standard-size units designated as affordable.

## Dimensional Standards Under Bonus
| Parameter | Micro-Unit Density Bonus Standard |
|-----------|----------------------------------|
| Maximum FAR | Base district FAR × 1.25 |
| Maximum height | Base district limit (no additional height) |
| Parking | 0.5 spaces per micro-unit |
| Unit size | Micro-units must not exceed 400 sq ft net rentable area |
| Setbacks | Base district standards |

## Required Plan Elements
1. **Cover sheet:** "Micro-Unit Density Bonus" stated; LDC §25-2-654 cited; total unit count; micro-unit count; non-micro unit count (if any)
2. **Project data table:** Total units, micro-unit count, affordable units, set-aside %, income limit (60% MFI), affordability period, tenure type
3. **Unit size exhibit:** Calculation or schedule showing net rentable area for each unit type; all units ≤400 sq ft labeled as micro-units
4. **Affordability schedule:** Each affordable unit by ID (must be micro-units), MFI tier, affordability period
5. **Site plan:** Parking count shown; height and FAR labeled; FAR calculation exhibit
6. **Floor plans:** Each unit dimensioned; micro-unit size confirmed; affordable micro-units labeled
7. **Elevations:** Maximum height shown

## Key Compliance Checks
- [ ] LDC §25-2-654 cited on cover sheet
- [ ] All or majority of units are micro-units (≤400 sq ft net rentable area)
- [ ] Net rentable area calculated correctly for each unit type (not gross, not including exterior walls)
- [ ] Affordable count ≥ 10% of total units — recompute from plan data
- [ ] Affordable units are themselves micro-units (≤400 sq ft)
- [ ] Income limit ≤ 60% MFI (stricter than other programs)
- [ ] Affordability period ≥ 40 years
- [ ] Proposed FAR ≤ base district FAR × 1.25 (show base district FAR and bonus calculation)
- [ ] Parking: 0.5 spaces per micro-unit (verify count matches unit count × 0.5, rounded up)
- [ ] No fee-in-lieu claimed (not available under this program)
- [ ] Affordable units individually labeled on floor plans

## Notes
- Net rentable area measurement: measured to inside face of perimeter walls, excluding closets if separately enclosed, but including bathroom and kitchen within the unit; balconies do not count
- Mixed projects (micro-units + standard units): FAR bonus applies to the whole project; affordable requirement is 10% of total units (micro and standard combined); but affordable units must still be micro-units
- The 60% MFI income limit is stricter than DB90, VMU, and most other programs — do not accept 80% MFI as compliant for this program
- Austin HCD may require a project-specific Regulatory Agreement; flag for [REVIEWER: verify Regulatory Agreement executed]
- This program does not grant height bonuses; any height increase requires a separate variance or combining district
